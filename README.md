# Org Link Battery

A little package that is an extraction of some core features from my setup-code which grew historical. I found it quite usefull and this package is an effort to make it useable by others.

It's a minimal version for org-links and some basic auto-completion. If you use [org-roam](https://github.com/org-roam/org-roam) or [org-brain](https://github.com/Kungsgeten/org-brain) this package is rather useless to you.

The package provides code that glues following packages together:
+ company-mode
+ org
+ org-id
+ bookmark+
